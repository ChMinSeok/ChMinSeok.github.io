def fac(n):
  if n == 1 :
    return 1
  else:
    return n * fac(n-1)

print(fac(10))

zip, for는 여러개가 가능함
```python
questions =['name','quest','color']
answers =['Kim','파이썬','blue']
for q, a in zip(questions, answers):
  print(f"What is your {q}? It is {a}")
# What is your name? It is Kim
# What is your quest? It is 파이썬
# What is your color? It is blue
```
